Run CPU install:
```bash
pip install -r requirements.txt
```
or GPU install (NVIDIA + CUDA 11.8):

```bash
pip install -r requirements-gpu.txt
```

Now, run:
```bash
image-to-math.py
```
or
```bash
math-to-image.py
```
or
```bash
pip install streamlit numpy pillow matplotlib
streamlit run viewer.py
```
